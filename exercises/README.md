# Exercises

Runnable examples linked from [TypeScript Blackbook](https://unional.github.io/typescript-blackbook/). Each topic folder includes a short README and optional `exercise.json` with URLs for StackBlitz, CodeSandbox, or similar.

| Topic | Description |
| ----- | ----------- |
| [node-this](./node-this/) | Node.js CommonJS: `this`, `module`, `exports`, and `global` |

The legacy file [`this.ts`](./this.ts) at the repo root is kept for convenience; the structured exercise lives under [`node-this/`](./node-this/).
